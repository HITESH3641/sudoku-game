# 🧩 Sudoku Solver

A modern, interactive Sudoku web application built with **HTML**, **CSS**, and **JavaScript**. The application fetches random Sudoku puzzles from a public API and provides features such as solving, checking answers, hints, and interactive board highlighting.

---

## 🚀 Features

* 🎲 Generate random Sudoku puzzles using a public API
* 📝 Interactive Sudoku board
* ✅ Check your solution with visual feedback
* 💡 Hint system for selected cells
* 🤖 One-click automatic solver with animation
* 🎨 Modern responsive UI
* 🔍 3×3 subgrid highlighting on hover
* 🌐 Automatic fallback puzzle if the API is unavailable

---

## 🛠 Tech Stack

### Frontend

* HTML5

## 🎮 How to Play

1. Click on an empty cell.
2. Click repeatedly to cycle through numbers **1–9**.
3. Press **CHECK** to verify your solution.
4. Press **HINT** to reveal the correct value for the selected cell.
5. Press **SOLVE** to watch the puzzle being solved automatically with animation.
